* Rails5.2.5

* Ajaxを使ったコメント機能

# カリキュラムの技術
***
* rails_admin
* cancancan
* faker   
# 実行手順
***
 ```
 $ git clone git@github.com:7sprinkle/beautician.git
 $ cd continue
 $ bundle install
 $ rails db:create && rails db:migrate
 $ rails s
```
